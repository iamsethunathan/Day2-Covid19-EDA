# COVID-19 Exploratory Data Analysis

This project performs Exploratory Data Analysis (EDA) on a country-wise COVID-19 dataset to understand patterns in confirmed cases, deaths, recoveries, and active cases.

## Objectives
- Analyze COVID-19 global statistics
- Understand distribution of key metrics
- Identify correlations between variables
- Visualize insights using charts

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains country-wise COVID-19 statistics including:
- Confirmed Cases
- Deaths
- Recovered
- Active Cases

## Analysis Performed
- Data overview and summary statistics
- Missing value analysis
- Distribution plots
- Correlation heatmap
- Country-wise death analysis

## Visualizations
The project includes multiple visualizations such as:
- Histograms
- Correlation heatmaps
- Distribution plots
